Thank you for downloading!

This mod changes the color of the Italian tanks from their default tan/yellow to a historical defensive grey-green color.
Included are skins for all Italian vehicles and matching icons for the garage and tech tree.

To install:
- Drop the "CWW-RecolorPack-Italian-Grigio-Difensivo" folder from inside the "MAIN MOD - COLOR CHANGE" folder into <Your WoT Directory>\mods\<game version number>. Example: C:\Games\World_Of_Tanks\mods\1.5.1.0

Optional:

- If you want mod support for the Dakka Improvement Project and/or my "Comprovato" remodel for the P.43 Ter, drag the "vehicles" folder from "ADDON - MOD SUPPORT, goes in res_mods version folder" into 
<Your WoT Directory>\res_mods\<game version number>. Example: C:\Games\World_Of_Tanks\res_mods\1.5.1.0.

Enjoy!
